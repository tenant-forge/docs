## Overview

Building a Sass platform is a complex task, but TenantForge makes it simple and easy to create a multi-tenant sass platform,
by providing a robust and secure solution for managing tenants, users, and permissions all throughout a fine tune and high configurable admin panel for your multi-tenant sass platform.

## TenantForge v1.x status

TenantForge v1.x is under active development. This means that it is currently in progress and is not yet ready for production use.

If you encounter any issues or have suggestions for improvements, please [open an issue on Github](https://github.com/tenant-forge/core/issues/new).
